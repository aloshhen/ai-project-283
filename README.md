NEO-AXIS ‚Äî THE MONOLITH PROTOCOL (AWWWARDS SOTY EDITION)
CORE CONCEPT: Create a hyper-sophisticated, interactive landing page for "NEO-AXIS" ‚Äî a next-generation neobank. The aesthetic is "Industrial Techno-Brutalism" and "Cyber-Minimalism," heavily inspired by Heliot Emil and high-end fashion archives. The site must look like a high-performance digital terminal with heavy WebGL integration and complex physics.

1. THE CENTRAL ARTIFACT (PHYSICS-BASED WEBGL):

Object: In the Hero section, place a giant, floating "Liquid Obsidian Iceberg" (a deforming 3D monolith).

Material: Reflective black chrome with high-index refraction and silver iridescent veins.

Cursor Repulsion Logic: Implement a non-Newtonian fluid shader. When the cursor moves near the monolith, the surface must "dent" and ripple away from the pointer in real-time.

Dynamic Fragmentation: Upon scrolling, the monolith should shatter into thousands of sharp geometric shards that rotate and reform into a credit card shape as the user reaches the next section.

2. VISUAL IDENTITY & SYSTEM:

Color Palette: Absolute Obsidian (#000000), Matte Charcoal (#0A0A0A), Liquid Silver, and "System-Alert Cyan" (#00FFFF) for micro-indicators.

Typography: * Headlines: Oversized, ultra-stretched Sans-Serif (e.g., Monument Extended). Use "Kinetic Weight" ‚Äî font weight changes from Thin to Black based on scroll position.

Data Logs: Micro-typography (8px - 10px) in Monospace (JetBrains Mono). Scatter technical metadata (latencies, coordinates, hash strings) in the corners of the viewport.

Atmosphere: Heavy film grain (noise), scanline overlays, and subtle "chromatic aberration" on the edges of the screen.

3. CUSTOM NEURAL CURSOR:

Visual: A "Tactical Crosshair" with a magnetic outer ring.

Interaction: The cursor displays live X/Y coordinates and "Scanning..." text next to the pointer.

Magnetism: When hovering over buttons, the cursor must "snap" to the element, and the outer ring must expand and rotate at 120fps. It should leave a temporary "glitch trail" behind its path.

4. LAYOUT & INTERFACE (BENTO 2.0):

Grid: An asymmetrical "broken" grid. Sections overlap with varying Z-index levels.

Components: * The Vault: A 3D safe-box that unfolds into a dashboard using "origami-style" animations.

Live Analytics: Real-time pulsing SVG waves that look like digital heartbeats or sonar pings.

Cards: Translucent "Glassmorphism" cards with brushed metal edges that tilt in 3D (Gyroscope/Mouse-tilt effect).

5. MOTION & TRANSITIONS:

Scroll: Implement "Inertial Momentum Scrolling" (buttery smooth).

Page Reveals: Content doesn't fade; it "assembles." Use "Shatter-reveal" or "Mask-wipe" animations where text blocks appear as if being decrypted.

Micro-interactions: Every click triggers a subtle screen shake or a "digital glitch" flicker.

6. TECHNICAL TAGS FOR AI LOGIC:

Style: Awwwards Site of the Year, FWA, CSSDA aesthetic.

Vibe: Cold, Elite, Secure, Industrial, Future-Proof.

Technologies: WebGL, Three.js, GSAP, Ray-tracing, Fragment Shaders, 4K Textures.
